# HR-ANALYTICS
HR data analysis using Power BI and Excel, showcasing attrition trends, department performance, and workforce demographics.
## Problem Statement
Organizations need to understand employee trends and HR metrics to make informed decisions. This dashboard helps HR teams analyze employee attrition, performance, and demographics effectively.
## Tools Used
- **Power BI**: For building interactive dashboard and visualizations  
- **Excel**: For data cleaning and preparation  
- **DAX**: For calculations and metri
## Dataset
- Employee data containing information on age, department, attrition, performance, years in comapny, training years,overtime, salary slab, marital status, job satisfaction, education field.  
- Dataset cleaned and prepared for analysis.

## Dashboard Features
- Employee attrition trends  
- Department-wise performance analysis  
- Workforce demographics (age, gender, tenure)  
- Interactive filters and slicers for dynamic insights
 ## Key Insights
- Highest attrition is observed by lab technicians and least in sales representatives. 
- Employees with average salary upto 5K have higher attrition. 
- Number of male employees have higher attrition than females.
  ## How to Use
1. Download the `HR_Analytics.pbix` file from this repository.  
2. Open it in Power BI Desktop.  
3. Explore the interactive dashboard using filters and slicers.
